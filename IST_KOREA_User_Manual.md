# IST KOREA 홈페이지 구축 완료 보고서 및 사용 설명서

## 1. 개요 (Project Overview)
본 문서는 IST KOREA의 기업 홈페이지 구축 프로젝트의 최종 구현 기능, 메뉴 구조 및 사용 방법을 설명하는 보고서입니다. 본 홈페이지는 최신 웹 기술을 적용하여 반응형 디자인, 다국어(국문/영문) 지원, 그리고 역동적인 사용자 경험(UX)을 제공하도록 설계되었습니다.

*   **기술 스택**: React, TypeScript, Vite
*   **스타일링**: Tailwind CSS
*   **애니메이션**: Framer Motion
*   **다국어 지원**: i18next (한국어/영어 완벽 지원)

## 2. 사이트 구조 (Sitemap)

홈페이지는 크게 사용자(Public) 페이지와 관리자(Admin) 페이지로 구분됩니다.

### 2.1 사용자 페이지 (Public)
*   **Home (메인)**
*   **Company (회사 소개)**
    *   인사말 (Greetings)
    *   연혁 (History)
    *   조직도 (Organization)
    *   오시는 길 (Location)
*   **Business (사업 영역)**
    *   Press Line (프레스 라인)
    *   Polishing Line (폴리싱 라인)
    *   Semiconductor Equipment (반도체 장비)
    *   Fiber Optics Equipment (광통신 장비)
*   **Product (제품 소개)**
    *   (Business 메뉴와 연동되어 각 사업 영역별 상세 제품 페이지로 이동)
*   **Support (고객 지원)**
    *   온라인 문의 (Inquiry) / 공지사항 등

### 2.2 관리자 페이지 (Admin)
*   **Dashboard**: 전체 현황판
*   **News Management**: 뉴스/공지사항 관리
*   **Careeres Management**: 채용 정보 관리
*   **Inquiries Management**: 고객 문의 관리

## 3. 주요 기능 및 화면 상세 설명

### 3.1 공통 요소 (Header & Footer)
*   **헤더 (Header)**:
    *   **로고**: 상단 좌측에 위치하며 클릭 시 메인으로 이동합니다.
    *   **네비게이션 (GNB)**: 마우스 오버 시 하위 메뉴가 펼쳐지는 드롭다운 방식을 적용했습니다.
    *   **다국어 토글 (Language Switcher)**: 우측 상단의 `KO / EN` 버튼을 통해 실시간으로 언어를 변경할 수 있습니다. 페이지 새로고침 없이 즉시 반영됩니다.
    *   **반응형 메뉴**: 모바일 환경에서는 햄버거 메뉴(≡)로 자동 변환되어 터치 친화적인 UI를 제공합니다.
*   **푸터 (Footer)**: 기업 정보(주소, 연락처, 대표자명) 및 SNS 링크, 저작권 표시가 포함되어 있습니다.

### 3.2 메인 페이지 (Home)
*   **히어로 섹션 (Hero Slider)**: 오토플레이 슬라이더를 통해 회사의 핵심 가치와 비전을 시각적으로 전달합니다. 각 슬라이드의 텍스트도 다국어 처리가 되어 있습니다.
*   **핵심 기술 (Tech Section)**: 회사가 보유한 주요 기술력을 아이콘과 함께 요약하여 보여줍니다.
*   **사업 개요 (Business Overview)**: 탭(Tab) 형식을 사용하여 주요 사업 분야를 한눈에 볼 수 있도록 구성했습니다.
*   **뉴스 티커 (News Ticker)**: 최신 소식을 흐르는 자막 형태로 노출하여 역동성을 더했습니다.

### 3.3 회사 소개 (Company)
*   하나의 페이지 내에서 **탭 네비게이션**을 통해 하위 콘텐츠(인사말, 연혁, 조직도, 오시는 길)를 전환하는 SPA(Single Page Application) 방식으로 구현되었습니다. 페이지 깜빡임 없이 부드럽게 콘텐츠가 전환됩니다.
*   **연혁 (History)**: 최신순으로 정렬된 타임라인 UI를 제공합니다.
*   **오시는 길 (Location)**: 카카오맵 등 지도 API 연동을 고려한 레이아웃과 주소 정보를 제공합니다.

### 3.4 사업 영역 (Business) - 핵심 기능
각 4개의 사업 영역(Press, Polishing, Semiconductor, Fiber)은 개별 페이지로 구성되어 있으며 다음과 같은 공통 기능을 가집니다:
*   **제품 네비게이션**: 페이지 상단(또는 좌측)에 해당 라인업의 제품들을 버튼 형태로 나열하여, 클릭 시 해당 제품 상세 정보로 즉시 전환됩니다.
*   **동적 데이터 바인딩**: 선택된 제품에 따라 이미지, 제품명, 설명, 사양(Specificatuon), 특징(Features), 옵션(Options) 데이터가 실시간으로 변경됩니다.
*   **데이터 관리**: 모든 텍스트 데이터는 번역 파일(`src/locales/*.json`)에서 관리되므로, 수정이 용이하고 다국어 확장이 쉽습니다.
*   **애니메이션**: 제품 전환 시 부드러운 페이드인/아웃 효과(Framer Motion)가 적용되어 고급스러운 느낌을 줍니다.

### 3.5 고객 지원 (Support)
*   사용자가 웹사이트를 통해 직접 문의를 남길 수 있는 폼(Form)을 제공합니다.
*   관리자는 관리자 페이지에서 접수된 문의 내역을 확인할 수 있습니다.

## 4. 관리자 모드 사용법 (Admin Guide)

관리자 페이지 접속 주소: `/admin/login`

1.  **로그인**: 지정된 관리자 계정으로 로그인합니다.
2.  **뉴스 관리 (News)**: 홈페이지 메인 티커 및 공지사항에 노출될 게시글을 작성, 수정, 삭제할 수 있습니다.
3.  **문의 관리 (Inquiries)**: 고객 지원 페이지를 통해 접수된 고객의 문의 사항을 리스트 형태로 확인하고 상태를 관리할 수 있습니다.

## 5. 유지보수 및 참고사항 (Maintenance)

### 5.1 텍스트 수정 (Localization)
화면에 표시되는 모든 문구는 소스 코드를 직접 수정할 필요 없이, 아래 JSON 파일만 수정하면 됩니다.
*   **한국어 파일**: `src/locales/ko/translation.json`
*   **영어 파일**: `src/locales/en/translation.json`

### 5.2 이미지 교체
이미지 파일은 `public/images/` 폴더 내에 위치합니다. 동일한 파일명으로 이미지를 덮어쓰면 서버 재시작 없이 반영됩니다.

### 5.3 주요 데이터 구조
Business 페이지의 제품 데이터는 `translation.json` 내의 `business_pages` 키 아래에 구조화되어 있습니다. 제품의 사양(specs)이나 특징(features)을 추가하려면 해당 JSON 배열에 항목을 추가하기만 하면 됩니다.
